The tool belt

Five verbs. One meeting.

The whole interface your agent drives — narrow enough to read in one screen.

$ samoagent join Bring a Recall.ai bot into a Zoom or Google Meet call. join
$ samoagent watch Stream the live transcript, one utterance per line. ears
$ samoagent frame Capture the current call view as a PNG, on demand. eyes
$ samoagent chat Send one message into the meeting chat. Explicit, never automatic. voice
$ samoagent leave Remove the bot and clean up local state. exit
also: status · transcript · dicts
What your agent gets

Eyes, ears, and a voice.

frame

Eyes

Captures the current call view as a PNG — only when your agent asks.

watch

Ears

Streams the live transcript, one timestamped utterance per line.

chat

Voice

Sends one deliberate message into the meeting — never on autopilot.

It looks only when you call frame. It speaks only when you call chat.

How it works

A real bot, driven locally.

A Recall.ai bot joins the call and streams the transcript and frames back to a local server your agent reads — over free ngrok HTTP, no TCP.

The default --ws-video frame path rides free ngrok HTTP — no TCP, no card verification.
Webhook and frame routes are token-protected; runtime files stay under ~/.samoagent/.
Need mixed video? An optional --rtmp path is there when you want it.
Your agent Decides when to join, watch, look, speak, or leave.
samoagent CLI join · watch · frame · chat · leave
Local server + ngrok Receives transcript and frame events over HTTPS / WSS.
Recall.ai bot Sits in the Zoom or Meet call and streams it back.
↑ streams back: transcript · chat · PNG frames
