# samograph.dev — Product Blueprint

This directory publishes the product spec for **samograph.dev**: a zero-setup,
hosted SaaS that puts a Recall.ai meeting bot in your calls and streams a
shareable, persisted live transcript.

## Contents

- **[SPEC.md](./SPEC.md)** — the **authoritative** specification (v0.2). This is
  the source of truth for agents and engineers building the product. Read this
  one for the full scope, architecture, user stories, test plan, and risks.
- **[brief.html](./brief.html)** — a polished, self-contained **human view**:
  a skimmable summary with architecture and user-journey Mermaid diagrams.
  Open it directly in a browser.
- **[TLDR.md](./TLDR.md)** — one-screen scope summary.

## Provenance

The spec was authored with [samospec](https://github.com/NikolayS/samospec) and
reached **v0.1** with one Claude review round (13 findings addressed). Full
multi-reviewer convergence is still **pending a working second reviewer seat**;
the document continues to evolve toward that convergence.
